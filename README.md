# PM Interview Coach

AI-powered PM interview analysis. Record or paste a transcript, get detailed coaching.

## Setup

1. Copy `.env.local.template` to `.env.local` and fill in your keys
2. `npm install`
3. `npm run dev`

## Keys needed
- `ANTHROPIC_API_KEY` — required for analysis
- `OPENAI_API_KEY` — required for audio transcription only
- `TURSO_DATABASE_URL` + `TURSO_AUTH_TOKEN` — from console.turso.tech
